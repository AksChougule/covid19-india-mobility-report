# SimCT

SimCT is a Python package for clinical study simulation workflows.
Phase 1 establishes foundational building blocks for loading YAML study blueprints,
canonical models, deterministic simulation scaffolding, and projection outputs.

## Phase 1 In Progress

Current scaffold includes:
- YAML blueprint loading entry points
- Core domain models and state/event placeholders
- Knowledge resolution and study initialization service stubs
- Deterministic simulation service skeleton
- DM and AE projector placeholders
- Pytest-based test skeletons
